MealForge Magic Link

Save recipes instantly from your browser. This works on sites that are usually hard to scrape.

Step 1: Copy this code

javascript:(function(){var t=encodeURIComponent(document.title);var u=encodeURIComponent(window.location.href);window.location.href='https://fmsohn.github.io/mealforge/?title='+t+'&url='+u;})();

Step 2

Create a new bookmark in your browser (any page will do).

Step 3

Edit the bookmark, name it "Save to MealForge", and paste the code above into the URL field.
